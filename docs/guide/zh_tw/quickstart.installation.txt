安裝
============

Yee 的安裝由如下兩步驟完成：

   1. 從 [yeeframework.com](http://www.yeeframework.com/) 下載 Yee 框架。
   2. 將 Yee 壓縮包解壓至一個 Web 可存取的目錄。

> Tip|提示: 安裝在 Web 目錄不是必須的，每個 Yee 應用程式都有一個入口腳本，只有它才必須暴露給 Web 用戶。其它 PHP 腳本（包括 Yee）應該保護起來不被 Web 存取，因為它們可能會被黑客利用。

需求
----

安裝完 Yee 以後你也許想驗證一下你的服務器是否滿足使用 Yee 的要求，只需瀏覽器中輸入如下網址來開啟需求檢測腳本：

~~~
http://hostname/path/to/yee/requirements/index.php
~~~

Yee 的最低需求是你的 Web 伺服器支援 PHP 5.1.0 或更高版本。Yee 在 Windows 和 Linux 系統上的 [Apache HTTP 伺服器](http://httpd.apache.org/) 中測試通過。理論上來說，在其它支援 PHP 5.1 的 Web 伺服器和平台上 Yee 都能正常運作。
