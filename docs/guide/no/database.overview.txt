Databaser
=========

Yee tilbyr avansert støtte for programmering mot databaser. Ved hjelp av
PHP Data Objects (PDO)-tillegget tilbyr Yee tilgang til ulike 
databasesystemer gjennom såkalte Data Access Objects (DAO).
Applikasjoner som benytter databaselaget til Yee kan enkelt bytte
database uten å endre koden som aksesserer den. Active Record (AR) i Yee
forenkler mappingen mellom tabeller i databasen og objekter. Ved å
representere tabeller som klasser og rader som instanser av klassen
eliminerer AR repetetive oppgaver som å skrive databasespørringer som
i hovedsak brukes til å oppdatere radene i tabellen.

Selv om DAO og AR i Yee kan håndtere nesten alle database-relaterte oppgaver
kan du fremdeles bruke dine egne databasebiblioteker. Yee er faktisk
designet for å brukes sammen med tredjepartskode.
