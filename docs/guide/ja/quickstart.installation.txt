インストール
============

Yee のインストールは主に以下の二つのステップからなります。

   1. [yeeframework.com](http://www.yeeframework.com/) から Yee フレームワークをダウンロード
   2. Yee リリースファイルをウェブアクセス可能なディレクトリへ展開

> Tip|ヒント: Yee 自体はウェブアクセス可能なディレクトリへインストールする必要はありません。
Yee アプリケーションはひとつのエントリスクリプトを持ちますが、通常、ウェブユーザに公開する必要があるのは、そのファイルだけです。その他の PHP スクリプトは、Yee のものも含めて、ウェブアクセスから保護されるべきです。そうしないと、ハッカーによって脆弱性を攻撃されるかもしれません。

必要条件
------------

Yee のインストールが終ったら、サーバが Yee の要求を満たしているかどうか確認するべきです。
以下の URL にある必要条件チェッカにウェブブラウザでアクセスすることで、それが確認できます。

~~~
http://hostname/path/to/yee/requirements/index.php
~~~

Yee は PHP 5.1 を要求します。従って、PHP 5.1 以上がサーバにインストールされており、ウェブサーバから利用できるようになっている必要があります。
Yee は Windows 上と Linux 上の [Apache HTTP サーバ](http://httpd.apache.org/) により検証されています。
PHP 5.1 がサポートされていれば、その他のウェブサーバやプラットフォームでも動作するでしょう。
