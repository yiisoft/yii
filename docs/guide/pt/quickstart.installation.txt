Instalação
============

A instalação do Yee envolve principalmente, as duas etapas seguintes:

   1. Fazer o Download  do Yee Framework em [yeeframework.com](http://www.yeeframework.com/).
   2. Descompactar o arquivo do Yee em um diretório acessível a Web.

> Tip|Dica: O Yee não precisa ser instalado em um diretório acessível a Web. Uma aplicação
Yee tem uma entrada de script que normalmente é o único arquivo que precisa ser exposto para
os usuários da Web. Outros scripts de PHP, incluindo os do Yee, devem ter o acesso protegido,
uma vez que podem ser explorados por hackers.

Requisitos
------------

Depois de instalar o Yee, você pode verificar se o servidor satisfaz todos os requisitos de 
de utilização do Yee. Você pode fazê-lo, acessando o script verificador de requisitos na 
seguinte URL em um navegador Web:

~~~
http://nomedoservidor/caminho/do/yee/requirements/index.php
~~~

O requisito mínimo para o Yee é que seu servidor Web tenha suporte ao PHP 5.1.0 ou acima. 
O Yee foi testado com [Apache HTTP Server](http://httpd.apache.org/) nos sistemas operacionais 
Windows e Linux. Também pode ser executado em outros servidores Web e plataformas desde que tenha o
PHP 5.
