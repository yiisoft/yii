עבודה עם מסד נתונים
=====================

Yee מספקת תמיכה נרחבת לתכנות ועבודה מול מסדי נתונים. Yee Data Access Objects (DAO) הבנויה על גבי התוסף של PHP הנקרא PHP Data Objects (PDO), מאפשרת גישה לסוגי מסדי נתונים שונים (DBMS) בממשק אחיד ונוח. אפליקציות המפותחות בעזרת ה DAO של Yee יכולות לעבוד בקלות עם DMBS אחרת ללא צורך בעריכת הקוד אשר ניגש למידע. Active Record (AR) אשר קיים ב Yee, מיושם באמצעות השיטה הנפוצה של Object-Relational Mapping (ORM), אשר מפשטת אף יותר את התכנות בעזרת מסד נתונים. ייצוג טבלה כמחלקה ושורה בטבלה כאובייקט, Yee AR מסירה את המשימה החוזרת על עצמה של כתיבת שאילתות SQL אשר ברוב המקרים מבצעות את אותו הדבר שהם פעלות CRUD (create, read, update, delete - יצירה, קריאה, עדכון ומחיקה) למיניהם.

למרות שמחלקות ה DAO ו AR במערכת ה Yee מטפלות בכמעט כל הנושאים הקשורים למסדי נתונים, תוכל עדיין להשתמש בספריה שלך המנהלת את מסד הנתונים באפליקציה. למעשה, Yee בנויה בצורה שתוכל לעבוד במקביל עם ספריות צד-שלישי נוספות.
