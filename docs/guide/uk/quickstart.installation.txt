Встановлення
============

Для встановлення Yee, як правило, необхідно виконати два кроки:

   1. Завантажити Yee Framework з [yeeframework.com](http://www.yeeframework.com/);
   2. Розпакувати реліз Yee в директорію, доступну із веб.

> Tip|Підказка: Насправді, Yee містить [вхідний скрипт](/doc/guide/basics.entry),
який звичайно є єдиним файлом з вільним доступом до нього із веб.
Інші PHP-скрипти, включаючи і файли Yee, повинні бути захищені від прямого доступу,
так як можуть бути використані для взлому.

Вимоги
----------

Після встановлення Yee перевірте, чи відповідає ваш веб-сервер необхідним
вимогам для використання Yee. Ви можете зробити це, скориставшись
спеціальним скриптом, доступним із веб-браузеру за адресою:

~~~
http://hostname/path/to/yee/requirements/index.php
~~~

Для роботи Yee вимагає як мінімум PHP 5.1. Yee тестувалася на
[Apache HTTP server](http://httpd.apache.org/) під Windows та Linux, але може
працювати також і на інших веб-серверах та платформах із підтримкою PHP 5.1.
