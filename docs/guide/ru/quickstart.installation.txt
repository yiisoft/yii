Установка
=========

Для установки Yee, как правило, необходимо выполнить два шага:

   1. Скачать Yee Framework с [yeeframework.com](http://www.yeeframework.com/).
   2. Распаковать релиз Yee в директорию, доступную из веб.

> Tip|Подсказка: На самом деле, Yee содержит [входной скрипт](/doc/guide/basics.entry),
который обычно является единственным файлом со свободным доступом к нему из веб.
Другие PHP-скрипты, включая и файлы Yee, должны быть защищены от прямого доступа,
так как могут быть использованы для взлома.

Требования
----------

После установки Yee проверьте, соответствует ли ваш веб-сервер необходимым
требованиям для использования Yee. Вы можете сделать это, воспользовавшись
специальным скриптом, доступным из веб-браузера по адресу:

~~~
http://hostname/path/to/yee/requirements/index.php
~~~

Для работы Yee требуется как минимум PHP 5.1. Yee тестировался на
[Apache HTTP server](http://httpd.apache.org/) под Windows и Linux, но может
работать также и на других веб-серверах и платформах с поддержкой PHP 5.1.
