Установка
=========

Установка Yii происходит в 2 шага:

   1. Скачайте Yii Framework с [yiiframework.com](http://www.yiiframework.com/).
   2. Распакуйте релиз Yii в директорию, доступную из веб.

> Tip: Yii не требует установки в директорию, доступную из веб.
Приложение Yii содержит входной (entry) скрипт, который обычно является единственным файлом, открытым для доступа из веб.
Другие PHP-скрипты, включая и файлы Yii, должны быть защищены от прямого доступа, так как они могут быть использованы для взлома.

Требования
----------

После установки Yii удостоверьтесь, что ваш сервер удовлетворяет всем необходимым требованиям для использования Yii.
Вы можете сделать это, воспользовавшись специальным скриптом, доступным из веб-браузера по следующему URL-адресу:

~~~
http://hostname/path/to/yii/requirements/index.php
~~~

Минимальные требования для Yii - поддержка вашим веб-сервером PHP версии 5.1.0 или выше. 
Yii тестировался с [Apache HTTP
server](http://httpd.apache.org/) на платформах Windows и Linux.
Скорее всего он будет работать с другими веб-серверами и на других платформах, поддерживающих PHP 5.
