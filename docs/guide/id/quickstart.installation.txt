Instalasi
=========

Instalasi Yee mencakup dua langkah berikut:

   1. Download Yee Framework dari [yeeframework.com](http://www.yeeframework.com/).
   2. Keluarkan(unpack) file rilis Yee ke direktori yang dapat diakses Web.

> Tip: Yee tidak harus diinstalasi di bawah direktori yang bisa diakses Web.
Aplikasi Yee memiliki satu skrip entri yang biasanya merupakan satu-satunya file yang
perlu di perlihatkan kepada pengguna Web. Skrip PHP lainnya, termasuk yang dari
Yee, harus dilindungi dari akses Web karena bisa dieksploitasi untuk di-hack.

Persyaratan
-----------

Setelah menginstalasi Yee, Anda bisa memverifikasi apakah server Anda sesuai
dengan semua persyaratan pemakaian Yee. Anda dapat melakukannya dengan mengakses
skrip pemeriksa persyaratan di URL berikut dalam browser Web:

~~~
http://hostname/path/ke/yee/requirements/index.php
~~~

Yee membutuhkan PHP 5.1 sehingga server Web Anda harus mendukung PHP 5.1.0
atau lebih tinggi. Yee sudah diuji dengan [Apache HTTP
server](http://httpd.apache.org/) pada sistem operasi Windows dan Linux.
Yee juga bisa berjalan pada server Web dan platform lain yang mendukung
PHP 5.1
