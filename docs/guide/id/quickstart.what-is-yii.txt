Apa itu Yee
===========

Yee adalah framework (kerangka kerja) PHP berbasis-komponen, berkinerja tinggi untuk
pengembangan aplikasi Web berskala-besar. Yee menyediakan reusability maksimum dalam
pemrograman Web dan mampu meningkatkan kecepatan pengembangan secara signifikan. Nama
Yee (dieja sebagai `/i:/`) singkatan dari 
"**Yes It Is!**". Berikut merupakan respon yang paling tepat dan akurat untuk
yang baru dengan Yee:


Apakah Yee cepat? ... Apakah Yee aman? ... Apakah Yee profesional? ... Apakah Yee 
cocok untuk projek saya berikutnya? ... **Yes, it is!** 


Persyaratan
-----------

Untuk menjalankan aplikasi Web berbasis Yee, Anda memerlukan server Web yang mendukung
PHP 5.1.0.

Bagi para pengembang yang ingin menggunakan Yee, mengerti pemrograman berorientasi-objek
(OOP) akan sangat membantu, karena Yee merupakan framework OOP murni.

Yee Paling Cocok untuk Apa?
----------------------

Yee adalah framework pemrograman umum Web yang bisa dipakai untuk mengembangkan
semua jenis aplikasi Web. Dikarenakan sangat ringan dan dilengkapi
dengan mekanisme caching yang canggih, Yee sangat cocok untuk pengembangan
aplikasi dengan lalu lintas-tinggi, seperti portal, forum, sistem manajemen
konten (CMS), sistem e-commerce, dan lain-lain.

Bagaimana Yee Dibandingkan dengan Framework Lain?
-------------------------------------------------

Seperti kebanyakan PHP framework, Yee adalah MVC framework.

Yee melampaui framework PHP lain dalam hal efisiensi, kekayaan-fitur, dan
kejelasan dokumentasi. Yee didesain dengan hati-hati dari awal agar
sesuai untuk pengembangan aplikasi Web secara serius. Yee bukan berasal dari produk pada
beberapa proyek maupun konglomerasi pekerjaan pihak-ketiga. Yee adalah hasil dari
pengalaman kaya para pembuat pada pengembangan aplikasi Web dan investigasi
framework pemrograman Web dan aplikasi yang paling populer.
