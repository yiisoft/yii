Ce este Yee
===========

Yee este o platforma PHP bazata pe componente de mare perfomanta pentru
dezvoltarea aplicatiilor Web de orice fel. Permite reutilizarea la maxim a codului scris
si poate accelera drastic procesul de dezvoltare. Numele Yee (pronuntat `/i:/`) inseamna *usor*,
*eficient* si *extensibil*.

Cerinte
-------

Pentru a rula o aplicatie bazata pe Yee, trebuie sa avem un server Web care pune la dispozitie minim
versiunea 5.1.0 de PHP.

Pentru programatorii care vor sa foloseasca Yee, intelegerea programarii orientata pe obiecte (POO)
ajuta foarte mult, pentru ca Yee este o platforma pura POO.

La ce foloseste Yee cel mai mult?
---------------------------------

Yee este o platforma de programare Web generica ce poate fi folosita pentru dezvoltarea
unei game foarte largi de aplicatii. Deoarece este foarte mica, si echipata cu
solutii de caching sofisticate, este folositoare in special pentru dezvoltarea
aplicatiilor cu trafic foarte mare: portaluri, forumuri, sisteme
de genstiune continut (CMS), sisteme e-commerce, etc.

Cum se compara Yee cu alte platforme PHP?
------------------------------------------

Ca cele mai multe platforme PHP, Yee este o platforma MVC.

Yee exceleaza, fata de celelalte platforme PHP, la capitolele eficienta, feature-uri si documentatie.
Yee a fost inca de la inceput foarte atent proiectata pentru a fi utila dezvoltarii
serioase de aplicatii Web. Nu este nici un rezultat al vreunui proiect, nici un conglomerat
de componente third-party. Este rezultatul experientei bogate a autorilor
in dezvoltarea aplicatiilor Web. In aceeasi masura este rezultatul unei investigari atente
ale celor mai populare aplicatii si platforme de programare Web.
