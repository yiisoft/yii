Instalación
===========

Para instalar Yee solo debe seguir los siguientes 2 pasos:

   1. Descargar el framework Yee de [yeeframework.com](http://www.yeeframework.com/).
   2. Descomprimir el archivo a un directorio accesible por el servicio Web.

> Tip: Yee no necesita ser instalado en un directorio accesible via web. La aplicacion
Yee tiene un script de entrada la cual usualmente es el único archivo que debe ser expuesto
a los usuarios Web. Otros scripts PHP , incluidos los de Yee, pueden (y se recomienda)
estar protegidos del acceso Web ya que esos pueden intentar ser explotado para Hackeo.

Requerimiento
-------------

Luego de instalar Yee, ustede puede verificar si su server satisface todos
los requerimientos para utilizar Yee. Para hacerlo debe hacer accesible
el script de verificación de requerimientos para utilizar Yee. Usted puede
acceder al script de verificación de requerimientos en la siguiente URL en
un explorador Web:

~~~
http://hostname/path/to/yee/requirements/index.php
~~~

El requerimiento mínimo de Yee es que su server soporte PHP 5.1.0 o superior.
Yee ha sido testeado con  [Apache HTTP server](http://httpd.apache.org/) en 
los sistemas operativos Windows y Linux. También puede funcionar en otras 
plataformas que soporten PHP 5.
