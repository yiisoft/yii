Cài đặt
=======

Cài đặt Yee đơn giản chỉ gồm 2 bước:

   1. Download Yee Framework từ [yeeframework.com](http://www.yeeframework.com/).
   2. Giải nén Yee vào một Web folder (folder trên web server có thể truy cập được bởi web server).

> Tip: Yee không nhất thiết phải được giải nén vào Web folder mà Web user có thể
truy cập được. Một ứng dụng Yee có một entry script (thông thường là index.php)
duy nhất là cần thiết phải được cấp quyền truy cập cho Web user. Các PHP file khác,
bao gồm cả mã nguồn của Yee (/framework) có thể được bảo vệ (bằng cách phân quyền)
để ngăn chặn bị harker tấn công.

Yêu cầu hệ thống
----------------

Sau khi cài đặt Yee, bạn có thể muốn kiểm tra xem Web server của bạn có đáp ứng tất
cả các yêu cầu về mặt cấu hình của Yee. Bạn có thể làm vậy bằng cách truy cập vào URL:

~~~
http://hostname/path/to/yee/requirements/index.php
~~~

Yêu cầu tối thiểu của Yee là Web server hỗ trợ PHP 5.1.0 hoặc version mới hơn. Yee đã
được test với  [Apache HTTP
server](http://httpd.apache.org/) trên Windows và Linux.

Yee cũng có thể chạy trên các Web server khác nếu có  hỗ trợ PHP 5.
