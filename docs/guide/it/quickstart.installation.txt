Installazione
============

L'installazione di Yee principalmente si sviluppa nei seguenti due passi:

   1. Scarica il framework Yee da [yeeframework.com](http://www.yeeframework.com/).
   2. Scompatta i file di Yee in una cartella accessibile dal web.

> Suggerimento: Yee non ha bisogno di essere installato in una cartella  
accessibile dal web. Una applicazione Yee ha un solo entry file che 
solitamente è l'unico file che deve essere esposto agli utenti web. Gli altri 
script PHP, compresi quelli di Yee, dovrebbero essere protetti dall'accesso 
dal web, altrimenti potrebbero essere sfruttati dagli hackers.

Requisiti
------------

Dopo l'installazione di Yee, si consiglia di verificare che il server 
soddisfi i requisiti di Yee. Si può fare accedendo allo script di controllo dei 
requisiti raggiungibile al seguente URL tramite un browser web:

~~~
http://hostname/path/to/yee/requirements/index.php
~~~

Yee richiede PHP 5.1, per cui il server deve avere PHP 5.1 o superiore 
installato e disponibile per il web server. Yee è stato testato con 
[Apache HTTP server](http://httpd.apache.org/) su Windows e Linux. 
Yee può funzionare anche con altri web server e piattaforme dotate di PHP 5.1.
