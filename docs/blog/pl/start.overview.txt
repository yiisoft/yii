Tworzenie bloga przy użyciu Yee
================================

Poradnik ten opisuje jak używać Yee aby stworzyć aplikację będącą blogiem znanej jako [demo blogu](http://www.yeeframework.com/demos/blog/), którą można znaleźć w poszczególnych wydaniach Yee. Wyjaśnia on w szczegółach każdy z kroków podejmowanych podczas dewelopmentu, który może znaleźć zastosowanie podczas tworzenia innych aplikacji webowych. Jako uzupełnienie [przewodnika](http://www.yeeframework.com/doc/guide/) oraz [opisu klas](http://www.yeeframework.com/doc/api/) Yee, poradnik ten ma na celu pokazać praktyczne zastosowanie, w miejsce szczegółowego opisu Yee. 

Od czytelników niniejszego przewodnika nie jest wymagana wcześniejsza znajomość Yee, aczkolwiek podstawowa wiedza o programowaniu zorientowanym obiektowo (OOP) oraz tworzeniu baz danych pomoże go im łatwiej zrozumieć.

> Note|Uwaga: Poradnik ten nie jest kompletnym przewodnikiem krok po kroku. Podczas przechodzenia przezeń będziesz musiał naprawiać pojawiające się błędy, sprawdzać API oraz czytać Przewodnik po Yee.

Ten przewodnik został wydany w zgodzie z [warunkami dokumentacji Yee](http://www.yeeframework.com/doc/terms/).
