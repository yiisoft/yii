Membangun Sebuah Sistem Blog dengan Menggunakan Yee
================================

Tutorial ini menjelaskan bagaimana menggunakan Yee untuk mengembangkan sebuah aplikasi blog seperti yang ditunjukkan pada [demo blog] (http://www.yeeframework.com/demos/blog/) yang dapat ditemukan pada setiap rilis Yee. Tutorial ini menjelaskan setiap detail langkah yang harus dilakukan pada pengembangan, yang juga bisa diaplikasikan pada aplikasi Web lainnya. Sebagai komplemen pada [Petunjuk](http://www.yeeframework.com/doc/guide/) dan [Referensi Kelas] (http://www.yeeframework.com/doc/api/) Yee, tutorial ini memfokus pada penjelasan penggunaan Yee bukan deskripsi mendalam dan padat.

Pembaca tutorial ini tidak perlu memiliki pengetahuan tentang Yee. Namun, pemahaman dasar konsep pemograman berbasis objek (OOP) dan pemograman basis data akan membantu pembaca untuk memahami tutorial ini lebih mudah.

Tutorial ini dirilis di bawah [ Aturan Dokumentasi Yee](http://www.yeeframework.com/doc/terms/).
