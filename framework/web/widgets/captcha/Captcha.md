`Captcha.ttf` font compiled from several third party open/free fonts.

# Files

* `Captcha.md`: description and notes, this file.
* `Captcha.ttf`: TrueType font file, binary.
* `Captcha.vfb`: FontLab Studio 5 project file, source.
